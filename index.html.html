1. 대사우관리란?

직원이 직접 사용하는 셀프서비스 메뉴입니다

대사우관리(ESS)에서 할 수 있는 것
메뉴 주요 기능
📋 개인정보변경신청 주소, 계좌번호, 연락처 등 내 정보 변경 요청
📄 증명서발급신청등록 재직증명서, 경력증명서, 원천징수영수증 등 발급 신청
🎁 복리후생신청 복리후생 항목 신청
✈️ 출장신청 출장 신청 및 내역 관리
🧾 세액조정신청 세액조정 신청
🌿 근태신청 연차, 반차, 반반차 등 휴가 신청
접속 방법 · ERP 로그인 후 왼쪽 메뉴에서 인사관리 → 대사우관리(ESS)를 선택하세요.
👤

2. 내 정보 변경 신청하기

인사관리 → 대사우관리(ESS) → 개인신청관리 → 개인정보변경신청

비활성화된 항목의 변경이 필요하신 경우, 담당자에게 개별 연락 부탁드립니다.
변경 신청 방법
1
개인정보변경신청 메뉴에 접속하면 현재 등록된 내 정보가 조회됩니다.
2
변경할 항목을 찾아 수정사항을 직접 입력합니다.
3
학력, 외국어, 경력, 자격면허, 여권비자, 증빙파일 내역에 항목을 추가하려면 우측 상단 '추가' 버튼을, 삭제하려면 '삭제' 버튼을 이용해 신청합니다.
4
입력 완료 후 우측 상단 '변경내역제출' 버튼을 클릭합니다.
5
'개인정보 변경신청내역' 버튼을 눌러 신청한 내역의 처리 상태를 확인할 수 있습니다.
신청 처리 상태 안내
상태의미다음 단계
상신 담당자가 아직 확인 전 담당자 검토 대기
종결 담당자가 승인하여 정보 반영 완료 내 정보에 즉시 반영됨
반려 담당자가 반려 처리 사유 확인 후 재신청 필요
승인취소 승인 완료 후 담당자가 취소 처리 담당자에게 문의 필요
💡 확인 방법 · '개인정보 변경신청내역' 버튼 → 내역 도움창에서 처리 상태 확인 가능
📄

3. 증명서 발급 신청하기

인사관리 → 대사우관리(ESS) → 개인신청관리 → 증명서발급신청등록

신청 가능한 증명서 종류
📋 근무경력증명서
  • 재직증명서
  • 경력증명서
  • 국문 / 영문 선택 가능
💰 소득증명서
  • 근로소득원천징수영수증
  • 근로소득원천징수부
  • 납세필증명서
증명서 신청 방법
1
증명서발급신청등록 메뉴에 접속합니다.
2
화면 상단에서 신청할 증명서 종류 아이콘을 클릭합니다.
(근무경력증명서 / 소득증명서 / 기타증명서)
3
신청 팝업에서 아래 항목을 입력합니다.
  • 증명서 종류 선택 (재직 / 경력 등)
  • 국문 / 영문 버전 선택
  • 주민등록번호 마스킹 여부 선택
  • 제출처 및 비고 입력
  • 신청 매수 입력
4
(결재상신) 버튼을 눌러 신청을 완료합니다.
5
담당자 승인 후 결재상태가 '종결'이 되면 출력 가능합니다.
목록 우측 프린터 아이콘을 눌러 출력하세요.
출력 유효기간 · 승인 완료일로부터 일정 기간 내에만 출력이 가능합니다. 기간이 지난 경우 재신청이 필요합니다. (정확한 기간은 담당자에게 문의)
🎁

4. 복리후생 신청하기

인사관리 → 대사우관리(ESS) → 개인신청관리 → 복리후생신청

신청 가능한 복리후생 종류
💐 경조사비
  • 결혼, 출산, 조의 등 경조 사유 발생 시 신청
🎓 학자금
  • 본인 또는 자녀 학자금 지원 신청
🏥 의료비 (건강검진)
  • 의료비 지원 및 건강검진 비용 신청
💳 대출
  • 사내 대출 신청
복리후생 신청 방법
1
복리후생신청 메뉴에 접속합니다.
2
화면 상단에서 신청할 복리후생 종류 아이콘을 클릭합니다.
(경조사비 / 학자금 / 의료비(건강검진) / 대출)
3
신청 팝업에서 해당 항목의 신청 내용을 입력하고, 필요 시 증빙 파일을 첨부합니다.
4
(결재상신) 버튼을 눌러 신청을 완료합니다.
5
신청 내역 및 처리 상태는 목록 화면에서 확인할 수 있습니다.
✈️

5. 출장 신청하기

인사관리 → 대사우관리(ESS) → 개인신청관리 → 출장신청

출장 신청 메뉴 안내
✈️ 출장신청
  • 국내·외 출장 신규 신청
✏️ 출장변경신청
  • 기신청한 출장 내용 변경
📝 복명신청
  • 출장 완료 후 결과 보고서 제출
출장 신청 방법
1
출장신청 메뉴에 접속합니다.
2
아래 항목을 빠짐없이 입력합니다.
  • 출장구분 (국내/해외), 출장시작일·종료일
  • 출장기관(명), 출장목적
  • 출장국가, 출발지, 도착지
  • 교통편, 차량번호
3
자차를 이용하는 경우, 우측 상단 '개인차량등록' 버튼을 눌러 차량 정보를 먼저 등록합니다.
4
하단 출장비정보 탭에서 '출장비계산' 버튼을 눌러 출장비를 계산합니다.
5
작성 완료 후 우측 상단 연필 모양(✏️) 결재 버튼을 눌러 상신합니다.
🧾

6. 세액조정 신청하기

인사관리 → 대사우관리(ESS) → 개인신청관리 → 세액조정신청

📌 세액조정이란? · 매월 급여에서 공제되는 세금의 공제율을 설정하는 기능입니다. 기본값은 100%로 설정되어 있으며, 필요에 따라 80% 또는 120%로 변경 신청할 수 있습니다.
세액조정 신청 방법
1
세액조정신청 메뉴에 접속합니다.
2
우측 상단 '신청' 버튼을 클릭합니다.
3
팝업 창에서 적용시작년월과 조정신청세율을 선택합니다.
세율은 80% / 100% / 120% 중 하나를 선택합니다.
4
신청사유를 입력한 후 하단 '결재' 버튼을 눌러 상신합니다.
유의사항 · 세액조정 신청한 세 방식은 과세기간 종료일까지는 변경할 수 없습니다.
🕐

7. 근태신청하기

인사관리 → 대사우관리(ESS) → 근태신청관리 → 근태신청

신청 가능한 근태 종류
🌙 연장근무
  • 평일 연장근무 신청
📅 휴일근무(특근)
  • 주말 및 휴일 특근 신청
🔄 보상휴가(대체휴무)
  • 특근에 대한 보상휴가 신청
근태 신청 방법
1
근태신청 메뉴에 접속 후 '근태신청서 신청' 버튼을 클릭합니다.
2
신청근태를 선택합니다.
  • 연장근무 (평일 연장)
  • 휴일근무(특근) (주말·휴일 근무)
  • 휴일근무(특근)_보상휴가 (대체휴무 신청 시)
3
신청기간 및 근무시간을 입력하고 신청사유를 작성합니다.
4
하단 '결재상신' 버튼을 클릭하면 전자결재 팝업이 뜹니다.
5
결재라인은 본인 상위 직급 결재라인에 맞게 설정하고, 여은정파트장을 수신 및 참조로 추가하여 신청을 완료합니다.
⚠️ 주의사항 · 신청 전 주 52시간 초과 여부를 반드시 확인하세요. 주 52시간을 초과한 경우 휴일근무(특근)으로 신청하더라도 담당자에 의해 보상휴가(대체휴무)로 전환될 수 있습니다.
🌿

8. 연차·휴가 신청하기

인사관리 → 대사우관리(ESS) → 근태신청관리 → 근태신청

휴가 신청 방법
1
근태신청 메뉴에 접속 후 '휴가신청서 신청' 버튼을 클릭합니다.
2
팝업 상단 신청구분에서 부여일수·사용일수·잔여일수를 확인합니다.
3
신청휴가와 기간을 선택합니다.
4
신청사유를 입력하고, 필요 시 첨부파일을 등록합니다.
5
하단 '결재상신' 버튼을 클릭하면 전자결재 팝업이 뜹니다.
6
결재라인은 본인 상위 직급 결재라인에 맞게 설정하고, 여은정파트장을 수신 및 참조로 추가하여 신청을 완료합니다.
취소 방법 · 아직 승인 전(대기 상태)인 신청 건은 해당 내역 선택 후 취소 처리가 가능합니다. 승인 완료 후에는 '휴가변경서 신청'을 이용해 주세요.
휴가 종류 안내
구분차감 일수비고
연차1일1일 단위 사용
반차0.5일오전/오후 선택
반반차0.25일2시간
외출차감 없음2시간 미만
조퇴차감 없음2시간 미만
기타 휴가-병가, 경조사 등 담당자에게 문의
보상휴가(대체휴무) 신청 방법
1
근태신청 메뉴에 접속 후 '휴가신청서 신청' 버튼을 클릭합니다.
2
신청휴가를 '보상휴가'로 선택합니다.
팝업 상단 신청구분에서 연차와 별개로 부여된 보상휴가 잔여일수를 확인할 수 있습니다.
3
사용할 기간을 선택하고 신청사유를 입력합니다.
4
하단 '결재상신' 버튼을 클릭하면 전자결재 팝업이 뜹니다.
5
결재라인은 본인 상위 직급 결재라인에 맞게 설정하고, 여은정파트장을 수신 및 참조로 추가하여 신청을 완료합니다.
⚠️

9. 주의사항 및 참고사항

사용 전 꼭 확인해 주세요

✅
신청 ≠ 즉시 반영
모든 신청은 담당자 승인 완료 후에만 실제 정보에 반영됩니다. 신청만으로 효력이 발생하지 않습니다.
🖨️
증명서 출력 유효기간
승인 완료 후 일정 기간 내에만 출력이 가능합니다. 기간 경과 시 재신청이 필요합니다.
🔒
개인정보 보호
증명서 출력 시 주민등록번호 마스킹 여부를 반드시 확인하고, 출력물은 안전하게 관리해 주세요.
문의처
문의 유형담당 부서비고
개인정보 변경 / 증명서 발급 여은정파트장 사내 인트라넷 또는 내선 문의
연차·휴가 신청 관련 여은정파트장 / 소속 팀장 승인 관련 사항은 소속 팀장님께 문의
시스템 오류 / 접속 불가 유승호 매니저님 ERP 관리자에게 문의
💬

10. 자주 묻는 질문 (FAQ)

사용 중 자주 겪는 상황들을 모았습니다. 질문을 클릭하면 답변이 펼쳐집니다.

개인정보 변경 관련
변경 신청을 했는데 언제 반영되나요?
신청 후 담당자가 승인을 완료해야 실제 정보에 반영됩니다. 처리 상태는 (개인정보 변경신청내역) 버튼에서 확인할 수 있습니다.
변경 신청을 취소하고 싶어요.
아직 미확인(승인 전) 상태라면 담당자가 승인취소 처리가 가능합니다. 이미 승인 완료된 경우에는 담당자에게 직접 문의해 주세요.
신청이 반려됐어요. 어떻게 하나요?
(개인정보 변경신청내역) 버튼에서 반려 사유를 확인한 후, 수정하여 다시 신청해 주세요.
연차·휴가 신청 관련
신청한 연차를 취소하고 싶어요.
아직 승인 전(대기 상태)인 경우 직접 취소 처리가 가능합니다. 이미 승인된 경우에는 휴가변경서 신청을 이용해 주세요.
내 잔여 연차가 몇 일인지 확인하고 싶어요.
휴가 신청 팝업 상단 신청구분에서 부여일수·사용일수·잔여일수를 확인할 수 있습니다.
증명서 발급 관련
출력 버튼이 활성화되지 않아요.
결재상태가 '종결'인 경우에만 출력이 가능합니다. 아직 담당자 승인 대기 중이거나 출력 유효기간이 지난 경우 출력이 불가합니다. 담당자에게 문의해 주세요.
영문 증명서를 신청하고 싶어요.
신청 팝업에서 영문을 선택하면 됩니다. 단, 사원정보에 영문 성명·주소가 미리 등록되어 있어야 합니다. 등록이 안 되어 있다면 담당자에게 먼저 요청하세요.
주민등록번호가 전부 보이게 출력되어 걱정돼요.
신청 팝업에서 주민등록번호 마스킹 적용을 선택하면 뒷자리가 가려진 상태로 출력됩니다.
다른 직원의 증명서를 대리 신청할 수 있나요?
같은 부서 내 상위 직급 인원이 하위 직급 직원을 대리하여 신청할 수 있습니다. 단, 타 부서 직원의 대리신청은 불가합니다.
출력 유효기간이 지났어요. 어떻게 하나요?
유효기간이 지나면 재출력이 불가합니다. 처음부터 다시 증명서 발급 신청을 진행해 주세요.
근태신청 관련
주 52시간 초과 여부는 어떻게 확인하나요?
근태신청 팝업에서 누계시간을 확인하면 됩니다. 그래도 잘 모르겠다면 담당자에게 문의해 주세요.
보상휴가(대체휴무)는 어떻게 사용하나요?
8번 연차·휴가 신청하기 메뉴에서 신청휴가를 '보상휴가'로 선택하여 연차와 동일한 방식으로 신청할 수 있습니다. 잔여 보상휴가 일수는 신청 팝업 상단 신청구분에서 확인하세요.
